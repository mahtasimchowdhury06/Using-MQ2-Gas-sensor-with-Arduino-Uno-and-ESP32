# Using-MQ2-Gas-sensor-with-Arduino-Uno-and-ESP32
A very common sensor in Robotics is MQ2 Gas sensor which can measure the gas level within it's surroundings. Using both an Arduino Uno and an ESP32, you can measure the level and able to know whether there is a harmful gas present or not.
You will need an Arduino Uno, an ESP32 and a MQ2 gas sensor to make this.
